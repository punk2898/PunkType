# 更新日志 / Changelog

本项目的所有重要变更都记录在此。版本遵循语义化版本。

---

## [1.4.1] — 2026-07-02

### 🐛 修复

- **越用越慢、要重启才恢复**：流式请求读到结尾后没有关闭底层连接，连接在会话中不断泄漏，攒满连接池后新请求排队变慢直至超时（重启 App 才清空）。现在读完即释放连接，并改用独立的、限制连接数的会话，和系统共享池隔离。这个问题所有人用久了都会遇到，强烈建议升级。

---

## [1.4.0] — 2026-06-21

新增「翻译」「询问」两个动作快捷键，修了几个影响体验的 bug，并改用**企业 Developer ID 签名 + 公证**（双击直接打开、零拦截）。

### ✨ 新功能

- **翻译键 ⌥⌘T**：口述 → 原意翻译成目标语言（默认中→英，设置里可改）→ 直接插入光标。
- **询问键 ⌥⌘A**：口述一个问题 → AI 回答，弹窗展示。
- 二者基于新的「快捷键 = 动作」骨架，后续可扩展更多动作。

### 🐛 修复

- **润色/极速变慢、"回不去"**：根因是残留的选中文字反复误触发命令模式（命令模式恒用 Pro 模型）。改为只认真实选中 + 记住上次目标，普通听写不再被卡进重处理。
- **润色会执行命令**：加了最高优先级铁律，口述内容只整理、绝不执行其中的指令。
- **格式档有时不排版**：App 感知的"聊天别过度排版"与格式档冲突 → App 感知改为只作用于润色档。
- 默认快捷键改回 **⌥Space**。

### 🧩 体验

- **首次启动权限引导窗**：三项权限实时状态 + 一键跳系统设置，授权后自动变绿（菜单栏「权限检查」可重开）。
- 移除冗余的「历史」设置页（记事本已全面接管）。
- 「关于」版本号改为自动读取，不再手动维护。

### 🔐 分发

- 改用**企业 Developer ID 签名 + 公证（notarized）**：同事下载双击即开，不再被 Gatekeeper 拦截。
- Bundle ID 调整为 `com.nexorainfinite.punktype`。

## [1.3.0] — 2026-06-19

这一版的主角是**记事本**——把你日常口述顺手攒成一份私密、本地的工作记录。

### 📓 记事本（新）

- **被动收录**：每次出字自动收进本地记事本，按天归档，附来源应用。零额外动作。
- **每日总结**：自动生成「日报」（标题 + 今天做了什么 + 重要性 + 思考/待办）。**启动时自动补**昨天及更早未生成的日报，第二天一开机就有。
- **窗口**：⌥⌘N 或菜单栏打开。左侧按天浏览、右侧日报 + 流水、全局搜索、条目可编辑/删除。
- **隐私排除**：默认收录所有应用，**微信默认不收录**；用过的应用出现在「收录范围」勾选列表，可随时排除。完全本地。

### 🩺 稳定性

- 新增可选的**诊断日志**（设置 → 关于；只记耗时不记内容），用于排查偶发卡顿。
- 流式输出的空闲超时 30s → 20s，缩短极端情况下的等待。

## [1.2.0] — 2026-06-17

这一版主打**更顺手**：换了更舒服的默认触发键、加了语音反馈，以及两个"更懂你"的智能功能。

### ⌨️ Fn 键触发（新默认）

- 默认快捷键改为 **🌐 Fn 键**：按一下开始、再按一下停止。Fn 是专用键，不和打字冲突，单击即可。
- 带防误触：按住 Fn 期间按了别的键（如 Fn+F 调音量）不会触发。
- 快捷键仍可自定义，原来的 ⌥Space 等组合都保留。

### 🔔 提示音

- 开始 / 停止录音各有一声轻提示音（音色不同，盲操作也能分辨）。
- 时序上避开了麦克风，不会被录进转写。可在设置关闭。

### 🧠 App 感知（按应用自动调语气）

- 自动识别你正在哪个应用输入，让 AI 贴合该场景：聊天软件口语化、邮件更正式、**代码 / 终端保留术语不乱改**、文档更书面。
- 默认开启，可在「设置 → 输出」关闭。

### 🎭 风格画像

- 可选地学习你的表达习惯（语气、句长、口头禅、标点、中英混用、称呼），润色时贴合，让输出越来越像你本人写的。
- 新增「个性化」设置页：可查看、手动编辑、重置。完全本地、仅自己可见，默认关闭。

### ⚙️ 默认值优化

- 三档默认改用 **Apple 快速识别（macOS 26）**，更快；macOS 25 及以下自动回落。
- 润色 / 格式档默认开启**流式输出**。

## [1.1.0] — 2026-06-15

这一版的主题是**更快**，外加把三档做成可自定义的预设，以及一批体验修复。

### 🚀 速度（核心）

- **关闭 DeepSeek 的"思考"**：`deepseek-v4` 系列默认会先推理再输出，对语音清理纯属浪费。现在对 DeepSeek 端点的清理/格式请求自动关闭思考，**首字从约 1.7 秒降到约 0.86 秒**（其它 OpenAI 兼容服务商不受影响）。
- **流式输出**（每档可开）：文字一边生成一边打进光标，不再等整段生成完才一次性粘贴。体感接近"说完就出"。
- **Apple 快速识别**（macOS 26）：可选接入 SpeechAnalyzer / SpeechTranscriber，本地、约 2.2× 快、无依赖；macOS 25 及以下自动回落到原识别引擎。
- **缩短停止后的等待**：识别封顶等待从 1.5 秒降到 0.7 秒，拿到最终结果仍立刻出字。

### 🎛️ 三档可自定义

- ⚡极速 / ✨润色 / 📄格式 现在是**三个可编辑的预设**，新增「档位」设置页。
- 每档可单独配：**识别引擎**（本机 / Apple 快速 / Whisper）、**AI 模型**、**流式输出**、**提示词**。
- 格式档不再强制走 Whisper 云端上传，默认改为本机识别 + Pro 模型。
- 保留智能默认值，同事装上零配置即可用。

### ✨ 体验

- **无光标时弹结果窗**：如果在 AI 处理时把光标点走了，结果会显示在弹窗里（可「插入到光标」或复制），不再悄悄丢进剪贴板。
- **更聪明的粘贴目标判断**：结合前台 App + 辅助功能焦点，减少误弹窗（终端 / Electron 等不暴露焦点的 App 也能正确粘贴）。
- **白天模式可读性修复**：录音浮窗不再用写死的白色，文字/波形随明暗自适应。
- **状态文案全中文化**：浮窗与菜单栏（聆听中 / AI 处理中 / 已粘贴 等）。
- **命令结果弹窗重做**：更大、可滚动、结果为主体，指令降为说明。

### 🛠️ 内部

- STT 引擎抽象为统一接口，新旧引擎可替换。
- 写死的档位流水线改为按配置驱动，更易维护。
